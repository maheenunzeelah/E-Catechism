E-Catechism
- Built a secure online testing platform focusing on student authentication and cheat prevention using the MERN stack and Machine Learning.
- Implemented facial recognition and GMM-based speaker recognition to verify student identity during exam registration and attempts.
- Developed a Gaussian Mixture Model (GMM) speaker authentication module by extracting MFCC features and training individual GMM models per student for accurate voice matching.
- Automated result generation for objective questions (MCQs, True/False).
- Developed CRUD functionalities for teachers and students using ReactJS.
- Built REST APIs using ExpressJS and used MongoDB as the NoSQL database.
